# Student Management System

A simple PHP and MySQL system for managing student records in XAMPP.

## Features

- Admin login
- Admin dashboard
- Student registration
- Add, edit, and delete students
- Search by name, admission number, email, or course
- Profile image upload
- MySQL database storage

## Setup

1. Start Apache and MySQL in XAMPP.
2. Open phpMyAdmin and import `database.sql`.
3. If needed, update the database settings in `config.php`.
4. Open the project in a browser.

Default admin account:

- Email: `admin@example.com`
- Password: `admin123`
